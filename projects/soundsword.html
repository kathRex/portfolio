// Creative Coding

Sound Sword

Hardware / Sensor Data / Motion Input / Real-Time Audio · 2023

Sound Sword is a motion-reactive cosplay prop that generates dynamic sound effects based on real sword movements. Using a Micro:bit controller and its built-in accelerometer, the system detects swings and translates them into synthesized audio in real time.

All audio is generated directly on the microcontroller using waveform synthesis. Different movement directions trigger different sound expressions and create a responsive interaction between physical performance and sound.

// Interaction System

Motion-Reactive Sound Design

The core interaction is based on the Micro:bit’s accelerometer, which continuously measures movement along the X, Y and Z axes. By repeatedly performing sword swings and analyzing the sensor output through a debugging interface, movement thresholds were calibrated for different swing directions.
When the sensor values fall within these calibrated ranges, the program triggers a matching sound expression. This allowed the sword to react differently depending on how it was moved, turning physical gestures into audio feedback.


Core Features

  • Motion detection through accelerometer data
  • Calibrated thresholds for multiple swing directions
  • Real-time sound response based on sword movement
  • Procedural audio generated directly on the microcontroller
  • Interaction designed for cosplay, performance and playful sword fights

// Technical Implementation

Embedded System & Physical Build

The physical prop was build by my partner, Estelle Schenker, and combines a handcrafted EVA foam sword with a removable Micro:bit-based electronics setup. The controller, speaker, battery pack and wiring were integrated into the prop while keeping the system accessible for recalibration and further code adjustments.
For better performance the project did not use prerecorded sound files and the audio had to be designed through mathematical waveform parameters.


Technical Highlights

  • Micro:bit embedded system with accelerometer input
  • JavaScript development in the MakeCode environment
  • Sensor data processing for movement classification
  • EVA foam prop construction and electronics integration
  • Removable controller setup for iteration and testing
